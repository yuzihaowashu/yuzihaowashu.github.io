# Zihao Yu - Personal Website

A clean, modern personal website showcasing research, publications, and academic achievements.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **Fast Loading**: Lightweight HTML/CSS with no external dependencies
- **Accessible**: Proper semantic HTML and focus management
- **Print Friendly**: Optimized styles for printing

## Structure

```
├── index.html          # Main website file
├── style.css          # All styling
├── files/             # Static files (CV, papers, etc.)
│   └── Yu_Zihao_CV.pdf
└── README.md          # This file
```

## Customization

### Personal Information
Edit the following in `index.html`:
- Name and title in the intro section
- Contact links (email, scholar, GitHub, CV)
- News items
- Publications list
- Research interests

### Styling
Modify `style.css` to change:
- Colors (update CSS custom properties)
- Typography (font families, sizes)
- Layout (spacing, container width)
- Animations and transitions

### Adding New Sections
1. Add HTML structure in `index.html`
2. Add corresponding CSS styles in `style.css`
3. Ensure responsive design for mobile devices
